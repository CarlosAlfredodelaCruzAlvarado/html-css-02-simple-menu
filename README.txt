Página de Login Simple - HTML y CSS

Este proyecto consiste en una página de login básica implementada con HTML y CSS. No se utilizó JavaScript ni React en este proyecto. La página no es responsive y no se incluyó funcionalidad de redirección al presionar el botón de login.

Contenido del Proyecto
index.html: Contiene la estructura HTML de la página de login.
index.css: Archivo CSS que proporciona el diseño y la apariencia visual de la página.

Uso
Clona o descarga el repositorio a tu sistema local.
Abre el archivo index.html en tu navegador web para ver la página de login.

Notas Adicionales
Este proyecto es un ejemplo simple de una página de login y no incluye funcionalidades avanzadas como autenticación o gestión de usuarios.
La página no es responsive y no se recomienda para dispositivos móviles.
No hay redirección después de presionar el botón de login. Es un proyecto estático con propósitos demostrativos.
